# DashTv - Live.

### Technology used 
Flutter and Dart.

1. For DB Storage    :- Shared Preference
2. For Navigation    :- Go router 
3. To check Internet :- internet_connection_checker_plus
4. Coding styple     :- MVVM & Modular Folder Structure
5. UI Design         :- Trending app resource - hotstar and peacock 
6. State Management  :- BLOC

## Features
1. Splash :- Splash Screen UI.
2. Homepage :- Shimmer Loading, Drawer, Trending Section, Now Playing Section, refresh indicator .
3. Details Page :- Movie detial with caching.
4. Explore Page :- Explore List with pagination and refresh indicator.
5. Bookmark Page :- Bookmark List.
6. Search Page :- Search List with debouncer.
